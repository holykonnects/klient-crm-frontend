# Google Apps Script Backends

This folder stores Google Apps Script source code used by the CRM frontend.

Each subfolder maps to a frontend module or shared backend. When a deployed Apps Script is pasted into Codex, place it in the matching folder as `Code.gs` or a clearly named `.gs` file.

Current folders include:

- `accounts` - Accounts module backend scripts.
- `auth` - Login/auth backend scripts.
- `calendar` - Calendar module backend scripts.
- `costing` - Costing and expense-request costing backend scripts.
- `deals` - Deals module backend scripts.
- `email` - Email templates, campaign, and logs backend scripts.
- `existence-search` - Existence search backend scripts.
- `inventory` - Inventory and stock management backend scripts.
- `leads` - Leads and lead form backend scripts.
- `orders` - Orders module backend scripts.
- `projects` - Projects module backend scripts.
- `quotations` - Quotation builder backend scripts.
- `sales-tracker` - Sales tracker backend scripts.
- `tenders` - Tender module backend scripts.
- `travel` - Travel module backend scripts.
